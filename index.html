Featured Work

TechMate — Voice AI Assistant

A real-time voice assistant built specifically for senior users — GCP Speech-to-Text streams audio input into a FastAPI inference layer that composes prompts for OpenAI, then pushes responses back through a React frontend. The behaviorally interesting piece is a KMeans clustering model that profiles each user's interaction patterns over time, feeding a custom context manager that prunes conversation history intelligently instead of sliding a fixed window.

React Python FastAPI GCP Speech-to-Text Firebase OpenAI KMeans
GitHub

agentic-lite — Multi-Model AI Chat

A general-purpose agentic UI that routes requests to OpenAI GPT or Google Gemini from a single Next.js interface, with Prisma-backed SQLite persistence keeping conversation history across sessions. The architectural challenge was a provider-agnostic streaming abstraction: both APIs emit tokens differently, and the React layer needed to handle partial updates without markdown or code block rendering flickering mid-stream.

Next.js TypeScript React OpenAI API Google Generative AI Prisma SQLite
GitHub

μTeX — Handwriting to LaTeX

A browser-only app that feeds a live camera stream into Google Gemini's multimodal vision API and returns LaTeX syntax for handwritten math equations — no server, no upload. Accessibility was a first-class requirement: the Web Speech API narrates equation output in real time so visually impaired users can verify transcriptions without switching tools. The entire stack is plain HTML/CSS/JS to keep deployment frictionless.

HTML CSS JavaScript Gemini Vision API Web Speech API MediaStream API
GitHub

Spring Boot Microservice + CI/CD Pipeline

A cloud-native Spring Boot service with a fully automated path from commit to production: a Jenkinsfile drives Maven builds, Docker image assembly, and Kubernetes rolling deployments in sequence without manual steps. The notable design choice was engineering zero-downtime releases via K8s readiness probes and a tuned RollingUpdate strategy — no service mesh required.

Java Spring Boot Maven Docker Kubernetes Jenkins
GitHub

Movie Script Summarization (NLP)

Implements the NAACL 2024 "Select & Summarize" pipeline: a RoBERTa-large scene saliency classifier first filters low-signal scenes from full movie scripts, then a fine-tuned Longformer Encoder-Decoder (LED) generates coherent summaries from only the retained content. The central finding — that pruning input beats feeding everything — holds even under ROUGE evaluation, which rewards recall, making selective compression a principled choice rather than a shortcut.

Python PyTorch RoBERTa Longformer (LED) HuggingFace NLP
