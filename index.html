Featured Work

TechMate: Voice AI Assistant

A real-time voice assistant built for senior users, combining GCP Speech-to-Text for audio capture with a FastAPI inference layer that composes prompts for OpenAI and streams responses to a React frontend. A KMeans clustering model profiles each user's interaction patterns over time, feeding a custom context manager that prunes conversation history based on behavioral clusters rather than a naive sliding window.

React Python FastAPI GCP Speech-to-Text Firebase OpenAI KMeans
GitHub

agentic-lite: Multi-Model AI Chat

A general-purpose agentic chat interface that routes requests across OpenAI GPT and Google Gemini from a single Next.js frontend, with Prisma-backed SQLite persistence for conversation history across sessions. The core engineering challenge was a provider-agnostic streaming layer: both APIs deliver tokens differently, so the React renderer needed a uniform buffering strategy to prevent markdown and code block flicker during partial updates.

Next.js TypeScript React OpenAI API Google Generative AI Prisma SQLite
GitHub

μTeX: Handwriting to LaTeX

A browser-only tool that captures handwritten math equations via the MediaStream API and passes frames to Google Gemini's multimodal vision endpoint, returning clean LaTeX syntax in real time with no server and no upload required. Accessibility was a first-class design requirement: the Web Speech API narrates each converted equation so visually impaired users can verify transcriptions without switching tools. The entire stack runs in a single HTML file.

HTML CSS JavaScript Gemini Vision API Web Speech API MediaStream API
GitHub

ShapeShift: 3D Generative Web App

A full-stack creative application pairing Next.js 15 with Three.js for real-time 3D visualization and Google Cloud Generative AI for content, with tRPC providing type-safe end-to-end API calls from client to a Prisma-managed database. The architectural challenge was integrating Three.js rendering into React's reconciler without triggering unnecessary re-renders, and streaming GCP Generative AI responses directly into the 3D scene graph.

Next.js 15 TypeScript Three.js tRPC GCP Generative AI Prisma Radix UI
GitHub Live Demo

MedSave: Medication Redistribution Platform

A marketplace connecting pharmacies holding surplus or near-expiry medications with nonprofits and community health clinics serving underinsured patients. The platform uses the NDC Directory API to auto-fill drug metadata from barcode input, Mapbox for proximity-ranked listings, and Google OAuth for role-based access that separates donor and recipient workflows, all backed by a Flask REST API with a React and Tailwind CSS frontend.

React Python Flask Tailwind CSS Google OAuth Mapbox NDC Directory API SQLite
GitHub

Spring Boot Microservice with CI/CD Pipeline

A cloud-native Spring Boot service with a complete automated delivery pipeline: a Jenkinsfile drives Maven builds, Docker image assembly, and Kubernetes rolling deployments from a single commit trigger. The key design decision was achieving zero-downtime releases through K8s readiness probes and a tuned RollingUpdate strategy, eliminating the need for a service mesh while maintaining safe deployments.
